Getting Help

On-Line Help

When DbSee is running, pressing the key F1  displays the Help Window.

Contextual Help is implemented to some extent - if the topic shown is not appropriate then use Contents or Search to locate the appropriate topic.


Off-Line Help

  • You can also display the Help file when DbSee is not running, by selecting Help from DbSee Program Menu Group.
    i.e Start ¢ Programs ¢ DbSee ¢ Help.
  • A User Manual which is the entire Help content in PDF format is also available for off-line reading and printing.  To access the Manual do Start ¢ Programs ¢ DbSee ¢ User Manual.


About

The Main Toolbar Menu Help ¢ About displays the About Splash Screen which has the Version Number of DbSee and contact links.
