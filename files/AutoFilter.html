Auto Filter

Auto Filter feature allows you to Filter the Grid Rows by one or more Column Values, very much like the feature in MS Excel.


Make sure that Load All Data option of the grid is turned on as explained in Grid Goodies.


  1. Open the Table Customers and customize its columns as illustrated in Columns Customize  so that column country is visible in the display area
  2. Click the Options button on the Grid Toolbar and check , if it is Unchecked. When this is checked a down arrow icon appears on every column header.
  3. Click the down arrow of header of column Country. On the pop-down select Canada. The grid is now filtered for Country = Canada and shows only those records that meets this condition.
    • Note that down arrow on the Header of Country column has turned blue indicating an active filter and the filter condition is displayed at the bottom of the grid.
    • To remove the filter, click the cross icon left of the filter condition displayed at the bottom.
    • You can apply multiple filter , by setting filter on other columns.
