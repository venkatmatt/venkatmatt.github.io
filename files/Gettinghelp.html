Installing DbSee

DbSee is distributed as Install Setup Program usually named DbSee_Install.exe ( but may contain version number in the name).


If you do not have it you can download it from my site

Save the downloaded file to a suitable place,  and double-click to run the Install Setup.


To run the Install Setup you must be logged into MS Windows as a user with Administrative privilege.


The Installation Setup is very simple and you can accept the defaults. It is strongly recommended that you do not change the Install Target Location unless you have some valid reason, such as lack of Disk Space.


At the end of Installation Setup, opt to Launch DbSee so that it starts. 


If any errors are encountered, check the Install Log file at

C:\Documents and Settings\All Users\Application Data\MattWare\DbSee\Log

for possible causes and remedy.
