Tables: Master-Detail

One of the best features of DbSee is its ability to automatically show data in Master/Detail relationships.

  1. Select the Tables tab if it is not already selected. Under this tab the Tables of the Database are displayed in a Tree List.
  2. Select the Customer table and click the Checkbox to the right of it - this opens the table and shows it data on the right under the Customer tab. The Tables tree also expands at Customer and displays its children Orders and Order Details.
  3. Now Check the Orders table to open it - its data is displayed under the Customer's data on the same tab. If you now click on any row of the Customers grid the Orders grid below will immediately display the data corresponding to the selected Customer.
  4. Now check the Order Details table on the Table tree - its data is now displayed to the right of the Orders grid, for the Order selected in the Orders grid.


We thus have a Master-Detail Related Data between Customers ¢ Orders  ¢ Order Detail
