Grid Toolbar

The Grid Toolbar situated just above the Data Grid provides a number of button to to control and manipulate the Grid.



Note that depending on the Options set in the DbSee's Options dialog, some buttons may not be visible. Some buttons are disabled depending on the status of the grid.

Grid Display Control

Button

Function

Auto Adjust Column width to fit display area. Useful when you have many columns. After Auto Adjust you can resize column width by dragging the vertical borders. This is a Toggle button. See Quick Demo / Grid Goodies for more info and illustration.

Customize Columns: clicking this displays the customize columns popup. Select the Columns you want to display and Unselect the rest. Rearrange the Column Display order by drag and drop. See Quick Demo - Columns Customize for more info and illustration

Change grid font - invokes standard Font Dialog that changes the Grid Font.

Displays Grid Options drop-down menu. The default values of these options can be set in DbSee Options.

Allow Edit and Edit on Click determines if the data can be edited and how - see Editing Table Data  for more information.  If the DbSee Option Read-Only is set then these two menu items are hidden.

The Load All Data to Grid must be enabled for the Auto Filter / Group / Sort features to work - see the relevant sections under Grid Goodies for more information on the Auto.. features.

Expands ALL Nodes when Grid data is grouped using the Auto Group feature or when the Data is displayed in a Tree Grid.

Collapses ALL Nodes when Grid data is grouped using the Auto Group feature or when the Data is displayed in a Tree Grid

Removes Grouping when Grid data is grouped using the Auto Group feature

Hides or Displays the Record View panel in which the current row is displayed in a two column (Column name & Value) grid. Record View is also used to display Lookup data.


Copy & Export

Copies All or Selected Rows to Clipboard. If more than one is selected then only the selected rows are copied else all rows are copied. You can paste the copied rows into another application as MS Excel, Word etc. To copy only the current row use the Right Click popup menu of the Grid.

Exports All or Selected Rows to a File. If more than one is selected then only the selected rows are exported else all rows are exported. In the Save Dialog that is displayed change the default file Name if necessary and select the File Type - MS Excel, Text, XML or HTML.


Data Editing: Visibility of these buttons depend on the Data Editing Options of DbSee

Adds a New Record after the last row of the Grid. Fill in the Data & click the Save Changes button to save or the Cancel Changes button to discard the newly added record.

Deletes the Current Record after displaying a confirmation. Note that the deleted record cannot be restored.

Saves Changes done to the current record in the Grid or Record View or Memo/Blob view. Moving to another record also posts the edits.

Cancel Edits done to the current record, if you have not moved away to another record.


Grid Popup (context) Menu

Right clicking on the Data Grid pops up a context menu

    • Use the Copy Cell Content to copy the content of the cell to the Clipboard
    • Use the Copy Row to copy the content of the entire row to the Clipboard.
