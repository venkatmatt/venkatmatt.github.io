Creating & Executing Queries

With DbSee you can

  • Create, Edit and Execute your own SQL Queries.  
  • Save them conveniently into a single file for later use.
  • Optionally Load a Queries File when DbSee Starts.


Queries are controlled from the user interface under the Queries Tab.

  • The Top Panel lists the Queries in the currently opened Queries File. It has a Toolbar buttons to Create New Queries File and Load and Save Queries File.
  • The Bottom Panel Shows the SQL Statement of the Query Selected on the Top Panel. It has 
    • An edit box to Edit the Name of the Query.
    • A button to Execute / Refresh the Query to display its data in the Data Area under a tab.
    • A button to Delete the currently selected Query.
