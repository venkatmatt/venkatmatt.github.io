Auto Group

Auto Group Feature allows you to group the Grid display by one or more columns, very much like in MS Outlook.


Make sure that Load All Data option of the grid is turned on as explained in Grid Goodies.


  1. Open the Table Customers and customize its columns as illustrated in Columns Customize, so that column country is visible in the display area.
  2. Click the Options button on the Grid Toolbar and check , if it is Unchecked. When this is checked a Grouping Panel is displayed above the Grid Header columns.
  3. Drag the Country column header into the Grouping Panel. The Grid Rows are now Grouped by Supplier ID.
    • Click on the + icon to expand any node to display rows belonging to that Supplier.
    • You can also Expand / Collapse all nodes by clicking the / button on the Grid Toolbar.
    • You can group by multiple columns by dragging more columns into the grouping panel.
