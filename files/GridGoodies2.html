Grid Goodies

The Data Grid in which the Table, View and Query Data are displayed has the following features, accessible through the buttons on the Grid Toolbar located just above the grid:


  • Auto Adjust Column Width
  • Columns Customize


  • Auto Group
  • Auto Filter
  • Auto Sort
  • Auto Search

The Auto Group, Filter and Sort features require that the Load All Data option of the Grid is turned on. Load All Data option consumes memory resources and hence is not recommended for Tables/Views/Queries with large number of records.

Load All Data can be enable in two different ways:

  • It can be set as Default in DbSee Options Dialog. Click the Options button on the Main Toolbar. On the Options Dialog select the Data tab. In the Data Grid group Check  Load All Data to Grid   on.
  • Or you can click the Options button on the Grid Toolbar and on the displayed popup Check
