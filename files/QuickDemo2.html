Quick Demo

After successful Installation you can launch DbSee, but to use it you need to connect to a Database.

To setup a database connection to the database of your choice may require some effort and help of your system administrator.

However you can easily explore the features of DbSee, by directly opening the sample Access database file created by the Install Setup.


On the Main Toolbar of DbSee click on Open Db File and select MS Access File


 

A Open Access Db File Dialog will be displayed and you should see the file Access Db Sample.mdb - select it and click Open


If you cannot see this file, click on the Look in Combo box at the top and change the folder to

C:\Documents and Settings\All Users\Application Data\MattWare\DbSee\Demo Data

This is where the Installation should have put the Demo Access File


When the File is successfully opened you can explore DbSee features

  • Master-Detail Data Display
  • Record View & Lookup
  • Memo & Blob View
  • Column Information
  • Grid Goodies
