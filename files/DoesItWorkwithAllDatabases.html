Does It Work with All Databases ?

DbSee uses Db Express Drivers or OLEDB Providers to access database. It is the availability and compatibility of the drivers/providers that determines if a database can be opened by DbSee.


DbSee has been Tested OK with the following Databases, with full Meta Data Extraction required for the Master/Detail and Lookup features:


  • Interbase Version 6.0.1.0 using Db Express Drivers (of 2004).
  • MSSQL Server Developer Edition Version 8.00.194, through both Db Express and OLEDB Provider.
  • Access 2000 through OLEDB Provider.


DbSee does not yet have Full Meta Data extraction capability for Oracle, DB2, Informix, MySQL, when accessed through Db Express drivers.

Full Meta Data extraction for these Databases may work when accessed through ADO (OLEDB Providers).

What this means is that when you connect to Databases other than Interbase, MSSQL or Access you may not have Master/Detail and Lookup features working.

You may also face problem with newer versions of any Database as the Db Express Drivers supplied with DbSee are 6 years old.
