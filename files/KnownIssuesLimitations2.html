Known Issues & Limitations

  1. Full Meta Data extraction and hence the Master-Detail and Lookup features are supported at present only for Interbase, MSSQL and Access Databases.
  2. Occasionally, DbSee goes into this crazy state in which pressing any key results in an Access Violation and displays an Error message. If this happens just close down DbSee and start again.
  3. The Record View does not handle BCD type data properly - it always show zero value for any BCD type data !
  4. Editing a Field's value in Record View immediately posts it, without giving an opportunity to Undo.
  5. Printing of data from DbSee is not supported - If required Export to Spreadsheet and Print.
  6. The Db Express drivers used are 6 year old and may not work with the latest versions of the Databases.
