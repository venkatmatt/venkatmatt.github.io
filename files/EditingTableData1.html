Editing Table Data

DbSee is not intended as a Data Editor but you can edit Table Data if you want.

View and Query Data are always Read Only and cannot be edited.


The DbSee Options and the Data Grid Options control if Data displayed in the Data Grid or Record View or Memo/Blob View can be Edited, Deleted and Added.


Editing

When Editing is enabled and the option Edit When Cell is Clicked set on, just click on the grid cell to edit its value otherwise press F2 or click the cell again to start editing. After editing the cell data press Enter to save or Esc to cancel.


  • The Grid data can also be edited in the Record View.
  • Memo data is edited in Memo View displayed below.
  • Blob/Picture data is edited or cleared using the right click menu


When editing is enabled the data grid and Memo has a white background color. When editing is disabled they have a light Yellow background color.


Moving away from the edited Record (row) automatically saves the data. You can also save data by clicking on the button on the Grid Toolbar. Before moving away from the record you cancel any changes you have made to the cells by clicking the button.


Adding a New Record

You can add new records if the DbSee option Allow Edit/Delete/Append is enabled.

Click the Grid Toolbar button  Add a New Record  to add a new blank record which is displayed after the last row.

  • Fill in the Column values for the newly added record either at the grid or at the Record View. If the Edit on Click option is not set then you need to press F2 before you can enter any value into a cell.
  • Note that you cannot edit Auto Incrementing Fields (Data Type = ftAutoInc) - when you save the newly added record it will automatically be assigned a unique value.
  • If you are entering the data in the grid cells, click on Save Changes button to save the data.


Deleting a Record

  • To delete a record select it on the Grid and click the Delete Record button.
  • At the Confirmation Dialog that is displayed select OK to delete the record.
  • Note that deleted records cannot be restored by button
