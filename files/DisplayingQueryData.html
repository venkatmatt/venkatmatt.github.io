Displaying Query Data

Before you can Execute a Query to Display its Data, you must have a query loaded.

  • You can either Load a previously created Queries File or
  • You can quickly enter an ad-hoc query in the default New File and New Query that DbSee displays when it starts-up and execute it immediately by clicking the Execute button.


To load a previously created and saved Queries File, click on the Open a Query File button on the top panel and select the Queries File in the displayed Open Dialog.

Only Queries saved from DbSee can be properly loaded. This is because DbSee uses a special format to save multiple queries in a single file.

  • If you have loaded a queries file, select the query you want.
  • Check the checkbox left of it or click the Execute button on the bottom panel to open the query.

If the query has any parameter(s) then DbSee displays a Enter Query Parameter dialog - Enter a value for each of the parameters (you cannot leave them blank) and click OK.


The query data is displayed in a grid, in the Data Display area on the right, under a new tab with the Query's name. The displayed data can be controlled and manipulated (as with Table and View data) as illustrated under Grid Goodies and Data Grid Features.
